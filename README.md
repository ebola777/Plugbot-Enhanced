# PlugBot Enhanced

PlugBot Enhanced is a bot for plug.dj.

It is a brand new design but still a pre-alpha version, many old features
such as emoticons are not included yet. I will add them back soon.

## Features

- Auto-woot
- Auto-queue
- SKip video
